## What is SentryBot?

SentryBot plays the daily grind of **Last War: Survival** for you —
collecting, training, healing, hunting zombies and joining every rally,
**around the clock**. You log in to spend power, not to farm it.

It's an external **pixel bot**: it only looks at the screen and moves the
mouse, like a player. It never reads or modifies the game's files,
traffic or memory — and it is built around one hard rule:
**it never spends a single gem.**

## Plays like a person, not a script

Bots get caught by their perfection. SentryBot was built deliberately
imperfect:

- **Curved, decelerating mouse** — never clicks the same pixel twice
- **Human schedule** — sleeps 7–8 h a night at a random time, takes lunch breaks
- **Human brake** — if the game shows a verification check, it stops and alerts you
